# API 文档

## 基础服务器地址

- 基础服务器地址：[https://apiv1.cakeip.com/openapi](https://apiv1.cakeip.com/openapi)

## 基础请求参数

- 基础请求参数：

```json
{
    "params": {},
    "appKey": "XXXXXXXXXXXXXXXXXXXXXXXXXXXXXX"
}
```

### 参数说明

<ResponseField name="appKey" type="string" required>
  appKey 密钥
</ResponseField>

<ResponseField name="params" type="object" required>
  参数集合
</ResponseField>

## 备注

- 所有请求都需要传 `appKey`。
